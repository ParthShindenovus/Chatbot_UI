Chat Widget Embedding Example

This is an example page showing how to embed the Chat Widget.

Embed Code

Add this script tag before the closing </body> tag:

<script
  src="https://your-cdn.com/widget/widget-loader.js"
  data-api-key="sk_live_abc123..."
  data-api-url="https://api.yourdomain.com"
  data-widget-url="https://your-cdn.com/widget"
></script>

Configuration

  • data-api-key: Your API key (required)
  • data-api-url: Backend API URL (optional)
  • data-widget-url: Widget CDN URL (required)

Widget Location

The chat widget will appear as a floating button in the bottom-right corner of the page.
